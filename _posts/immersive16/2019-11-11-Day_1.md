## 1. Javascript runtime & Node.js
- 런타임이란 프로그래밍 언어가 구동되는 환경을 의미
    - 어떤 프로그래밍 언어가 동작할 수 있는 프로그램
- Javascript 런타임
    1. 브라우저
    2. Node.js
- 즉, 런타임이란 프로그래밍이 돌아가는 환경, Node는 Javascript 런타임

## 2. What is NVM
- NVM은 *Node Version Manager*의 줄임말
- NVM을 통해 간단한 명령어로 Node를 설치하고 관리 가능

```
$ nvm use (사용하고 싶은 Node version : use 명령어 전에 해당 node version이 설치되어있어야 합니다.)
ex) nvm use 12.13.0
```

## 3. What is Package.json
- Package.json에는 이 프로그램을 실행시키기 위해 필요한 모듈은 무엇인지, 프로그램을 실행시키는 방법, 테스트 방법 등이 명시
- 제품을 설명하는 카달로그와 같은 역할
