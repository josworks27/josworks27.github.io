# Bubble Sort
> 주어진 정렬되지 않은 배열을 오름차순으로 버블정렬 하는 함수를 작성하는 문제

```javascript
bubbleSort([2, 1, 3]); // yields [1, 2, 3]
```

## 접근법
1. 반복문을 이용하여 0번째 인덱스(왼쪽)부터 시작하여 바로 옆 인덱스의 값과 비교하여 큰 수를 오른쪽으로 이동시킴
2. 배열의 길이만큼 모든 수를 비교하여야 하며, 이를 배열의 원소만큼 반복해줘야 함
3. 반복해줄 때 마다 비교할 원소의 수가 줄어들어야 함. (이 부분을 보완해야 함)
4. 큰 수가 항상 오른쪽으로 위치하게 되기 때문에 오름차순 배열이 됨

## 자바스크립트 코드

```javascript
var bubbleSort = function (array) {
  for (let j = 0; j < array.length; j++) {
    for (let i = 0; i <= array.length; i++) {
      let temp = 0;
      if (array[i] >= array[i + 1]) {
        temp = array[i];
        array[i] = array[i + 1];
        array[i + 1] = temp;
      }
    }
  }
  return array;
};

```
