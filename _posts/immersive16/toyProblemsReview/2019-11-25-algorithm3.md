# Is subset of
> 어떠한 배열이 주어졌을 때, 주어진 배열의 원소와 비교하는 배열의 원소를 비교하여 비교하는 원소의 배열이 주어진 배열의 부분집한인가를 확인하는 문제

```javascript
var a = ['commit','push']
a.isSubsetOf(['commit','rebase','push','blame']) // true

var b = ['merge','reset','reset']
b.isSubsetOf(['reset','merge','add','commit']) // true 
```

## 접근법
1. 반복문을 이용하여 만들어진 인스턴트와 주어진 배열의 원소를 비교함
2. 인스턴트의 i번째 원소가 주어진 배열에 없을 경우 flag을 true에서 false로 변경하
3. flag이 false라면 부분집합이 성립되지 않기 때문에 false를 반환함

## 자바스크립트 코드

```javascript
Array.prototype.isSubsetOf = function(array){
  let copyArr = array.slice(0);

  let flag = true;
  for(let i = 0; i < this.length; i++) {
    if (!copyArr.includes(this[i])) {
      flag = false;
    }
  }
  return flag;
};
```
