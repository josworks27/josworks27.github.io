# nth Fibonacci
> 인자로 n(type: number)이 주어질 때, 피보나치 수열의 n번째 값을 찾는 문제

```javascript
피보나치 수열의 예시
0 1 1 2 3 5 8 ...

nthFibonacci(2);  // 1
nthFibonacci(3);  // 2
nthFibonacci(4);  // 3
```

## 접근법
1. 피보나치 수는 마지막 두 수를 더한 값이 다음 수가 되는 원리임
2. 효율적 메모리 활용을 위해 재귀방식을 이용하지 않고 while 반복문을 활용함
3. temp라는 임시변수를 활용하여 값을 임시로 저장하고, 마지막 두 수를 더한 값을 그 자리에 넣어줌
4. temp에 임시로 저장되어있던 수는 그 앞에 넣어줌
5. while 반복문을 통해 주어진 n만큼 반복함

## 자바스크립트 코드

```javascript
var nthFibonacci = function (n) {
  var a = 1
  var b = 0
  var temp;

  while (n >= 1) {
    temp = a;
    a = a + b;
    b = temp;
    n--;
  }
  return b;
};
```
