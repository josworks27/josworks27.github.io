# RockPaperScissors
> 어떤 사람이 가위바위보를 할 경우 낼 수 있는 모든 경우의 수를 고려하는 문제

```javascript
[
 ["rock", "rock", "rock"],
 ["rock", "rock", "paper"],
 ["rock", "rock", "scissors"],
 ["rock", "paper", "rock"],
         ...etc...
]
```

## 접근법
1. value가 특정한 수를 인자로 받지 않는다면 디폴트 파라미터를 이용하여 세 번으로 기본값 지정함
2. 같은 수를 계속 내는 중복되는 경우도 고려하여야 하며, 모든 경우의 수를 출력할 수 있어야 함
3. 재귀함수를 이용하였으며 count가 0인 경우 탈출조건으로 만들고, 빈 배열인 combi에 concat 메서드를 활용해 추가하는 방식으로 모든 경우의 수를 배열로 만들어 줌

## 자바스크립트 코드

```javascript
var rockPaperScissors = function (value) {
    let arr = ['rock', 'paper', 'scissors'];
    let result = [];

    function recursion(combi, count = 3 ) {
        if (count === 0) {
            result.push(combi)
            return;
        }
        for (let i = 0; i < arr.length; i++) {
            recursion(combi.concat(arr[i]), count - 1);
        }
        };
    recursion([], value)
    return result;
};
```
