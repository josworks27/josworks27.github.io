# Non-repeated Character
> 어떠한 문자열이 주어졌을 때, 같은 문자열 내에 반복되지 않는 첫 번째 문자를 찾아내는 문제

```javascript
 firstNonRepeatedCharacter('ABA');  // 'B'
 firstNonRepeatedCharacter('AACBDB');   // 'C'
```

## 접근법
1. 반복문을 이용하여 i번째 인덱스의 문자가 i 이후의 문자열에 있는지 확인함
2. slice 메서드를 이용하여 비교하는 문자열을 만들어 줌
3. 조건문의 조건을 비교하는 i번째 문자가 slice된 문자열에 없으며, 지나간 문자열 중에도 없을 경우 result라는 변수에 문자를 계속 더해준 후 반환함

## 자바스크립트 코드

```javascript
var firstNonRepeatedCharacter = function (string) {
  if (string.length === 0) null;

  let storage = '';

  for (let i = 0; i < string.length; i++) {
    let result = '';
    let slicedStr = string.slice(i + 1);

    if (slicedStr.includes(string[i]) === false && storage.includes(string[i]) === false) {
      result += string[i];
    }
    if (result !== '') {
      return result;
    }
    storage += string[i];
  }
};
```
