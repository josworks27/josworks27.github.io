# Common Characters
> 인자로 주어지는 문자열들을 비교하여 모든 문자열에 같은 문자가 있는지 찾는 문제

```javascript
commonCharacters('acexivou', 'aegihobu')  // 'aeiou'
commonCharacters('qwerty', 'wertyu', 'ertyui', 'rtyui'
      , 'tyuiop', 'yuiopa');  // 'y'
```

## 접근법
1. rest parameter의 0번째 인자를 기준으로 함
2. 0번째 인자인 문자열의 각 문자를 반복문으로 접근하여 나머지 인자들의 문자열에 문자들과 비교함
3. 나머지 문자열에서 같은 문자가 없다면 flag변수를 false로 변환함
4. flag이 true인 즉, 모든 문자열에서 해당 문자가 있는 문자만 result에 더해주고 반복문이 끝나면 반환해줌

## 자바스크립트 코드

```javascript
var commonCharacters = function (...param) {
  let firstParam = param[0];
  let others = param.slice(1);
  let result = '';

  for (let i = 0; i < firstParam.length; i++) {
    let flag = true;

    for (let j = 0; j < others.length; j++) {
      if (!others[j].includes(firstParam[i])) {
        flag = false;
      }
    }
    if (flag) {
      result += firstParam[i];
    }
  }
  return result;
};
```
