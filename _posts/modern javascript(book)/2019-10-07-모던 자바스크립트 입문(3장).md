# 3.1 변수

## 3.1.1 변수
* 변수: 값을 담기 위해 이름을 붙인 상자

## 3.1.2 변수 선언
* 선언자: var, let, const
* 자바스크립트는 변수 타입이 없기 때문에 모든 데이터 타입의 값을 저장할 수 있음
* 변수를 선언하기만 하면 **undefined** 값이 들어감

## 3.1.3 변수 선언 생략
* 변수를 선언하지 않고 사용하는 행위는 버그의 원인이 될 수 있음
* Strict 모드를 이용하여 선언하지 않은 변수를 찾아낼 수 있음

## 3.1.4 변수 끌어올림과 변수 중복 선언
* 변수 선언의 끌어올림(Hoisting): 프로그램 중간에서 변수를 선언해도 변수가 프로그램 첫머리에 선언된 것 처럼 다른 문장 앞에 생성됨
* 단, 선언과 동시에 대입하는 코드는 호이스팅 X

```javascript
console.log(x);  // undefinded
var x = 5;
console.log(x); //  5
```

## 3.1.5 변수의 명명 규칙
* 변수, 함수, 라벨 이름 등 사용자가 정의하는 이름을 **식별자**라고 함
  1. 사용할 수 있는 문자는 알파벳(a ~ z, A ~ Z), 숫자, 밑줄(_), 달러 기호($)가 있음
  2. 첫 글자로는 숫자를 사용할 수 없음
  3. 예약어를 식별자로 사용할 수 없음
* **캐멀 표기법**, 파스칼 표기법, 밑줄 표기법 등이 있음
* 자바스크립트는 주로 캐멀 표기법을 사용(예: createNewName)

# 3.2 데이터 타입

## 3.2.1 데이터 타입과 변수의 동적 타이핑
* 데이터 타입: 숫자나 문자열처럼 변수에 저장하는 데이터 종류
* 자바스크립트는 같은 변수에 숫자나 문자열과 같은 다양한 타입의 데이터를 대입할 수 있음
* 변수에 저장된 데이터 타입을 동적으로 바꿀 수 있는 언어를 **동적타입 언어(Dynamic typed language)** 라고 함

## 3.2.2 데이터 타입의 분류
1. 원시타입: 숫자, 문자열, boolean, undefined, null, symbol
  * 데이터를 구성하는 가장 기본적인 요소로 불변 값으로 정의됨
2. 객체타입: 배열, 함수, 정규 표현식 등
  * 변수 여러 개가 모여서 만들어진 복합 데이터 타입임
  * 객체타입의 값을 변수에 대입하면 변수에는 그 객체에 대한 참조(메모리에서의 위치정보)가 할당됨

## 3.2.3 숫자
* 자바스크립트는 타입이 없기 때문에 숫자를 모둘 64비트 부동소수점으로 표현함
* Infinity(플러스 무한대), NaN(부정 값, Not a Number)

## 3.2.4 문자열
* 자바스크립트의 문자열은 길이가 16비트인 유니코드문자(UTF-16 코드)를 나열한 것으로 전 세계에서 사용하는 문자를 대부분 표현할 수 있음
* 문자열 리터럴은 작은따옴표(')나 큰따옴표(")를 문자열 앞뒤에 붙여서 표현함
* \t, \n 등의 이스케이퍼 시퀀스 사용가능함  * 본 책 83페이지 참고

## 3.2.5 논리값
* 논리값은 조건식이 true인지 false인지를 표현하기 위해 사용하는 값임
* 논리값은 주로 제어구문(if/else문, while문, do/while문, for문)에서 사용함

## 3.2.6 특수한 
* 값이 없음을 표현하기 위한 특수한 값에는 null, undefinedrk 있음
* undefined: 정의되지 않은 상태
  1. 값을 아직 할당하지 않은 변수의 값
  2. 없는 객체의 프로퍼티를 읽으려고 시도했을 때의 값
  3. 없는 배열의 요소를 읽으려고 시도했을 때의 값
  4. 아무것도 반환하지 않는 함수가 반환하는 값
  5. 함수를 호출했을 때 전달받지 못한 인수의 값
* 변수 값이 undefined가 되는 것은 **값을 할당하지 않은 결과**임
* 코드로 undefined를 대입하는 것이 아니라 자바스크립트 엔진이 변수를 undefined로 초기화한 것
* null: 아무것도 없는 상태
* null은 주로 프로그램에서 무언가를 검색했지만 찾지 못했을 때 아무것도 없음을 전달하기 위한 값으로 사용됨

# 3.3 ECMAScript 6부터 추가된 데이터 타입

## 3.3.1 심벌(Symbol)
* 심벌은 자기 자신을 제외한 그 어떤 값과도 다른 유일무이한 값임

### 심벌의 생성

```javascript
var sym1 = Symbol();
```

## 3.3.2 템플릿 리터럴
* 템플릿 리터럴을 사용하면 표현식의 값을 문자열에 추가하거나 여러 줄의 문자열을 표현할 수 있음

### 기본적인 사용법
* 템플릿 리터럴은 역따옴표(`)로 묶어서 사용함  * 본 책 88페이지 참고

### 보간 표현식
* 템플릿 리터럴 안에는 플레이스 홀더를 넣을 수 있으며, ${...}로 표기함
* 자바스크립트 엔진은 플레이스 홀더 안에 든 ... 부분을 표현식으로 간주하여 평가(evaluation)함

```javascript
var a = 2, b = 3;
console.log(`${a} + ${b} = ${a + b}`);  // 2 + 3 = 5
```
